// Capabilities

What it does

A fully containerized pipeline that ingests, transforms, and warehouses financial data from multiple free sources.

⚡

Real-Time Streaming

Kafka producer fetches live 1-minute bars from Yahoo Finance for all 10 tickers. Market-hours aware with concurrent fetching.

📊

Batch ETL Pipeline

Airflow DAGs run daily incremental ETL per ticker. 25 years of historical data on first load, then only new data.

🏦

Star Schema Warehouse

9-table TimescaleDB warehouse with dimensions (company, date, macro) and facts (prices, fundamentals, earnings, SEC filings).

📈

Fundamentals & Earnings

Daily company metrics (market cap, PE, beta), quarterly EPS surprises, and SEC 10-K/10-Q financial statements via EdgarTools.

🌎

Macro Data

FRED macro indicators — federal funds rate, CPI, unemployment, GDP — contextualize stock performance against the broader economy.

📦

Fully Containerized

7 Docker services orchestrated via Compose. One command to start: TimescaleDB, Airflow, Kafka, Zookeeper, producer, consumer.

Quickstart

Up and running in under a minute. Requires Docker and Docker Compose.

# Clone and start git clone https://github.com/jayhemnani9910/stock-data-platform.git cd stock-data-platform cp .env.example .env # edit with your FRED API key docker compose up -d # Access Airflow UI open http://localhost:8081 # admin / admin # Schema auto-creates. Trigger DAGs in order: # 1. populate_dim_company # 2. populate_dim_date # 3. etl_stock_data_aapl (or any ticker) # Query the warehouse docker exec -it timescaledb psql -U data226 -d stockdw \ -c "SELECT * FROM fact_stock_price_daily ORDER BY date DESC LIMIT 10;"
